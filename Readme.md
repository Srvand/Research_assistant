InfoGenie: Your Capital Market Document Analysis Partner
Info Genie is a user-friendly application designed to empower your market research by streamlining the analysis of capital market documents. It leverages the power of Langchain and multimodal Rag techniques to extract text, images, and tables from PDFs, allowing you to summarize key findings and get answers to your specific questions with ease.

Key Features:-

Intelligent PDF Processing: Extracts text, images, and tables with high accuracy.
Comprehensive Analysis    : Provides summaries of the extracted data for quick understanding.
User-Focused Q&A          : Answer your specific questions directly from the analyzed document.
Intuitive Interface       : Streamlit-based interface for a user-friendly experience.

Target Users:-
1) Market Researchers
2) Investment Analysts
3) Financial Professionals

Getting Started:-
Prerequisites             : Ensure you have Python 3.x and pip installed on your machine.
Installation              : Clone this repository and run pip install -r requirements.txt to install required library dependencies.
Run the App               : Navigate to the project directory and run streamlit run app.py. This will launch the InfoGenie app in your web browser.

Usage:-
1) Upload a capital market document (PDF format) through the file upload interface.
2) Explore the extracted data (text, images, tables) presented within the app.
3) Utilize the Q&A section to ask your specific questions about the document's content. InfoGenie will leverage its analysis to provide relevant answers.

Disclaimer:-
Info Genie is intended as a tool to assist with market research and analysis. It should not be solely relied upon for investment decisions.
